# BasicUtils基础的工具类集合

## 一、简介及使用说明
#### 1.1 概述
这是一个基础的工具类集合，有一些地方还不是很完善。

#### 1.2 简介
目前主要包含的工具类共有以下几个:<br>
- 1.利用ExecutorService实现的异步线程池工具类<br>
- 2.数据值计算工具类<br>
- 3.日期工具类<br>
- 4.利用HttpClient进行post请求的工具类<br>
- 5.图片压缩工具类<br>
- 6.价格计算工具类<br>
- 7.uuid生成工具类(对 java.util.UUID工具类的简单封装,todo)
